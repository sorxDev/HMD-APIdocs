# HowManyDrifters? API Docs

[Punda](https://discord.com/users/398355997929897986) built [howmanydrifters.com](https://howmanydrifters.com) and the API behind it is really cool, so I put together this documentation to make it easier for people to code their own projects with it.

Everything here was mapped out by exploring the live API. Endpoints, response shapes, parameters, all of it. If you want to build something on top of average Orion Drift data, this should get you started.

**[View the docs](https://sorxdev.github.io/HMD-APIdocs/)**
